## 零钱通项目

模拟钱包存入，取出，查询记录

### 程序入口

Main

### 功能展示

* 初始界面

```
============Menu===========
1. 明细          
2. 收入          
3. 消费          
4. 清除          
5. 退出   
```

```

2022-03-07 16:38收入100<br>
2022-03-07 16:38收入340<br>
结余540<br> 
```

```

```

* 取出
